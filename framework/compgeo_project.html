Pedagogical Aid for Procedural Map and Terrain Generation

Procedural generation of terrain has long been useful in games and simulations. Using techniques such as delaunay triangulations and Voronoi diagrams, developers can generate realistic looking terrains for use in their applications. The techniques that are used for generating terrains are also useful for reconstruction of maps in 3-D space and simulating naturally occuring phenomena such as earthquakes and erosion. Therefore for this project, I plan on creating an application/pedagogical aid that will allow users to see the breakdown of procedural terrain generation process and see how it takes place. My plan is to create a tool that not only generates a random terrain but allows users to observe the process and modify certain variables so that they can gain a better understanding of this application of Computational Geometry.
